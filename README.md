# 📦 Pembukuan Pesanan

Sistem pembukuan pesanan berbasis PHP & MySQL dengan tampilan modern iOS 26 Glass UI.

## ✨ Fitur Utama

- Login Username (Role Admin & User)
- Manajemen Produk
- Harga Jual & Harga Modal
- Perhitungan Keuntungan Otomatis
- Input Pesanan Pelanggan
- Status Pembayaran
- Status Pengiriman
- Riwayat Pesanan
- Export Excel
- Export PDF
- Dashboard Penjualan
- Grafik Penjualan Mingguan
- Grafik Penjualan Bulanan
- Pendapatan 7 Hari
- Pendapatan 30 Hari
- Pendapatan Semua Waktu
- Filter Custom Tanggal
- ACC Pembukuan Per User
- Hapus Pesanan Per User
- Hapus Pesanan Per Tanggal
- Hapus Semua Pesanan
- WhatsApp Integration
- PWA Ready (Install seperti aplikasi Android)

## 📊 Dashboard Admin

Admin dapat:

- Menambah User
- Menambah Produk
- Mengedit Produk
- Menghapus Produk
- Melihat Riwayat Semua User
- Melihat Total Pendapatan
- Melihat Total Keuntungan
- Export Laporan Excel/PDF
- ACC Pembukuan User

## 👤 Dashboard User

User dapat:

- Membuat Pesanan
- Mengubah Status Pembayaran
- Mengubah Status Pengiriman
- Melihat Riwayat Pesanan
- Export Riwayat Excel/PDF
- Menghubungi Pelanggan melalui WhatsApp

## 🛠️ Teknologi

- PHP 8+
- MySQL
- Bootstrap
- Chart.js
- PWA
- HTML5
- CSS3
- JavaScript
